一、项目概述
1.1 项目名称
智连词境（AI Word Connect Game）
1.2 项目定位
一个基于 AI 内容生成与智能教学反馈 的网页端词语连连看学习游戏，通过游戏化机制提升用户对词汇（成语 / 英文单词）的理解、辨析与记忆能力。
1.3 适用场景
- 教育类 / AI 类 / 文化类比赛
- 学校课程作业 / 作品集
- 教育产品 Demo
1.4 核心价值
- 用 AI 解决词汇学习中“死记硬背、缺乏反馈”的问题
- 将传统连连看升级为“可教学的智能游戏”
--------------------------------------------------------------------------------
二、玩法与功能设计
2.1 基础玩法
- 游戏形式：词语连连看（≤2 拐点消除）
- 棋盘规模：4×4 / 6×6 / 8×8（随难度变化）
- 匹配规则：
- 中文：成语 ↔ 正确释义
- 英文：单词 ↔ 中文释义 / 英文释义
2.2 可选学习模式
模式说明中文成语成语 + 释义 + 文化背景英文词汇单词 + 含义 + 例句主题模式学习 / 品德 / 成长 / 科技难度分级初级 / 中级 / 高级
--------------------------------------------------------------------------------

--------------------------------------------------------------------------------
四、核心业务流程
4.1 游戏流程
1.玩家选择：语言 / 难度 / 主题
2.系统请求 AI 生成词语关卡数据
3.Board 模块随机铺牌
4.玩家进行连连看操作
5.系统判定匹配结果
6.AI 给出解释 / 提示
7.游戏结束 → 学习总结
--------------------------------------------------------------------------------
五、AI 功能设计（比赛核心）
5.1 AI 职责拆分
模块AI 作用内容生成动态生成词语与释义难度控制根据表现调整复杂度学习讲解给出对错原因学习总结输出复习建议
--------------------------------------------------------------------------------
5.2 AI Prompt 设计（示例）
你是一名语言教学专家。
请生成 6 组用于“词语连连看”游戏的学习卡片。

要求：
1. 语言：{{language}}
2. 难度：{{level}}
3. 主题：{{theme}}
4. 给出：词语、准确释义、易混淆词、学习提示

请只输出 JSON 数组。

--------------------------------------------------------------------------------
六、接口文档（核心）
6.1 获取关卡词语
POST /api/words/generate
请求参数
{
  "language": "zh | en",
  "level": "easy | medium | hard",
  "theme": "learning"
}

响应示例
[
  {
    "word": "循序渐进",
    "meaning": "按照一定顺序逐步推进",
    "hint": "与一蹴而就含义相反",
    "confuse": "一蹴而就"
  }
]

--------------------------------------------------------------------------------
6.2 提交对局数据（可选）
POST /api/game/summary
{
  "correct": 8,
  "wrong": 3,
  "duration": 120
}

七、核心逻辑说明
7.1 连连看算法逻辑
- 使用 BFS 搜索路径
- 判断规则：
- 直线连接
- 一个拐点
- 两个拐点
算法复杂度低，适合实时判断
--------------------------------------------------------------------------------
7.2 动态难度逻辑
条件调整策略连续正确增加格子 / 干扰项多次错误降低难度 / 增加提示时间过长启用高亮提示
--------------------------------------------------------------------------------
八、数据结构设计
interface WordCard {
  word: string
  meaning: string
  hint: string
  confuse?: string
  level: number
}
